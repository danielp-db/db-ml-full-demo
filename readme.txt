# READ ME

This is a quick demo showcasing some of the Machine Learning  capabilities of Databricks within a workspace.

The topics are:
- Quick pipeline to ingest data and transform into features
- Quick pipeline to train a model, evaluate it and deploy it for batch scoring and monitor it with Data Quality.
- Notebook to deploy the model for Near-Real Time Inference and monitor it with Inference Tables.
- Detour for features like
  - AutoML 
  - Playground
  - AI 
- Quickly show `AI` functions.
- Build a quick Agent that uses RAG with Vector Search / Structured Data and functions.
- Discussion about GenAI Evaluation tools.
